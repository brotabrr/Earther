## how to use script:

go to https://python.org  
download it for your operation system and make sure its 3.x.x  
open the installer file and read carefully, you should see at bottom at somepoint "install python to PATH"  
MAKE SURE ITS CHECKED!!  

verify it worked by opening command prompt (press windows key and type "cmd")  
and then type "`python --version`"  
if you fucked up something, use google, or AI if you are lazy  


## once you have python installed:   
place "Python.py" and your 3d model into folder in desktop (or whenever you want)  
then make sure that on Python.py the last lines config matches your 3d model name  
double click the python file (or open via running `python Python.py` on your cmd if youre not dumb)  
and it should make Lua file  
place that lua file in your executors workspace folder  
and run "Roblox.lua" when you are in planecrazy  
if your game lags, use lower poly 3D model
